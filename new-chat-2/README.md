# 블로그 표지 이미지 제작 앱

업체명, 강의명, 대상자, 강사명, 사진을 입력하면 1:1 블로그 표지 이미지를 제작할 수 있는 정적 웹앱입니다.

## 주요 기능

- 1080 x 1080 정사각형 표지 이미지 생성
- 사진 업로드 후 컬러 배경 이미지로 사용
- 상단 업체명 라벨, 큰 강의명, 강조 문구, 대상자 문구, 강사명 배지 표시
- 강조 컬러 선택
- PNG 또는 JPG 다운로드
- 브라우저에서 바로 실행 가능

## 로컬 실행

별도 설치 없이 `index.html` 파일을 브라우저로 열면 됩니다.

로컬 서버로 확인하려면 아래 명령을 사용할 수 있습니다.

```bash
python -m http.server 8787
```

브라우저에서 `http://localhost:8787`로 접속합니다.

## Netlify 배포

이 프로젝트는 빌드가 필요 없는 정적 사이트입니다.

Netlify 설정:

- Build command: 비워두기
- Publish directory: `.`

`netlify.toml`에 아래 설정이 포함되어 있어 Netlify가 자동으로 현재 폴더를 배포합니다.

```toml
[build]
  publish = "."
```

## 사용 방법

1. 업체명을 입력합니다.
2. 강의명을 입력합니다. 줄바꿈을 넣으면 두 번째 줄이 강조 문구로 표시됩니다.
3. 대상자 또는 보조문구를 입력합니다.
4. 강사명을 입력합니다.
5. 사진을 업로드합니다.
6. 강조 컬러와 다운로드 형식을 선택합니다.
7. 다운로드 버튼을 눌러 PNG 또는 JPG 파일을 저장합니다.
